Leftovers
Leftovers is a recipe application that allows you to take some shortcuts when looking to cook leftovers or entire meals. You don't have to wade through lengthy Author stories, or plow through advertisments and cookie notifications. This app allows you to view required ingredients of meals before ever having to visit the website. It also has several filter methods that let you filter out recipes for multiple health criteria. It lets users set up favorite recipe lists with a drag and drop feature. This was designed for both desktop and mobile.
You can find the Git repo for leftovers here
Taskmaster Pro
Taskmaster Pro is a task program that creates tasks that the user and manipulate into different catagories with a drag and drop feature. It uses several light weight packages that make it very accessible to users. Touchpunch UI gives the JQuery the ability to be interpreted on mobile. Media Queries make this application work both on desktop and mobile
You can find the Git repo for Taskmaster here
